# Differentiation

## Category: AEO Content System

We don't compete on writing. We compete on **whether AI agents cite your brand**. This is a new category — most competitors don't know it exists yet.

## Citation Architecture (Named Methodology)

Six structural patterns baked into every post that AI agents use to decide what to cite:

1. **Explicit capability statements** — Not marketing implications. Direct, extractable claims agents can quote verbatim.
2. **Hierarchical H2→H3 heading structure** — Every H2 mirrors a question an agent would receive. First sentence is the direct answer.
3. **Comparison tables** — Structured, parseable data. When a user asks "X vs Y," agents pull your table as the definitive answer.
4. **FAQ sections mapping to AI query patterns** — Conversational Q&A pairs matching how users actually ask agents.
5. **Branded concept anchoring** — Citable named frameworks (like "Citation Architecture" itself). Gives agents a unique term to attribute to your brand.
6. **Use-case-driven examples** — Concrete scenarios matching the queries agents receive. Agents prefer specificity over abstraction.

This methodology is the product. Everything else (the pipeline, the scheduling, the cost) is delivery mechanism.
