# Social Distribution

Amplify blog content by sharing to social platforms. All posts require user approval before publishing.

## Approval Workflow

**Never post without confirmation.** Always:
1. Draft the social post
2. Send the draft to the user via Telegram (or show in REPL)
3. Wait for explicit approval before posting
4. Post only after receiving confirmation

## Platform Playbooks

### Reddit

Best subreddits for our content:
- **r/LocalLLaMA** -- Technical posts about local AI, model comparisons
- **r/selfhosted** -- Self-hosting guides, infrastructure posts
- **r/artificial** -- Broader AI industry pieces
- **r/commandline** -- CLI tool posts, developer workflow content
- **r/opensource** -- Open-source project announcements, community posts

**Posting guidelines:**
- Read subreddit rules before posting (check sidebar/wiki)
- Use descriptive titles, not clickbait
- Self-promotion rules: most subreddits limit self-promotion to ~10% of your posts
- Prefer text posts with value upfront, link to blog at the end
- Best times: weekday mornings US time (9-11am ET)
- Engage with comments after posting

**Title formats that work:**
- "I built X to solve Y" (Show & Tell)
- "Guide: How to do X with Y" (Tutorial)
- "X vs Y vs Z: My experience after N months" (Comparison)

### X / Twitter

**Thread format for blog posts:**
1. Hook tweet: Key insight or surprising stat from the post
2. 3-5 tweets expanding on main points
3. Final tweet: Link to full post + CTA
4. Keep each tweet under 280 chars, use line breaks for readability

**Single tweet format:**
- Lead with the insight, not the link
- Use 1-2 relevant hashtags max (#AIAgents, #OpenSource, #DevTools)

### Hacker News

**Submission tips:**
- Title: Use the exact blog post title (HN penalizes editorialized titles)
- Best times: 6-9am ET weekdays
- "Show HN:" prefix for product launches or demos
- Don't ask for upvotes -- it's against HN rules
- Be ready to engage in comments (HN rewards participation)

## Cross-Linking

After publishing social posts:
- Add social links back to the blog post as a comment or update
- Link between platforms ("Discussion on HN: [link]", "Reddit thread: [link]")

## Draft Templates

### Reddit Post Draft
```
Title: [descriptive, non-clickbait title]

[2-3 sentence summary of the key insight]

[Main value from the blog post -- don't just tease, provide substance]

Full post: [blog URL]

---
Happy to answer any questions!
```

### Twitter Thread Draft
```
1/ [Hook: surprising stat or key insight]

2/ [Problem statement]

3/ [Solution/approach]

4/ [Key result or benefit]

5/ Full breakdown on the blog: [URL]
```

### HN Submission Draft
```
Title: [exact blog post title]
URL: [blog post URL]
```
